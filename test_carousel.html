IBM Watson Assistant Setup Instructions

  1. Create a Dialog Node in your Watson Assistant
  2. Add a Response with type "JSON Editor"
  3. Use the JSON structure shown above
  4. Set response_type to "user_defined"
  5. Add user_defined_type as "carousel"
  6. Include carousel_data array with your items
